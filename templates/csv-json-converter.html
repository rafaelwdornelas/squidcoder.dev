Como usar:

CSV para JSON: Cole seus dados CSV no campo de entrada e clique em "Converter" para obter o JSON equivalente.

JSON para CSV: Cole seus dados JSON no campo de entrada e clique em "Converter" para obter o CSV equivalente.

Opções de CSV: Selecione o delimitador correto e indique se a primeira linha é um cabeçalho.

Opções de JSON: Escolha entre JSON formatado ou minificado, e se deseja incluir colchetes de array.
