Sobre o Código Octal

O código octal é um sistema de numeração de base 8, que usa os dígitos de 0 a 7. Na computação, o octal é frequentemente usado como uma forma mais compacta de representar valores binários, já que cada dígito octal corresponde exatamente a três dígitos binários.

Na programação, especialmente em linguagens como C, Python e Bash, os números octais são muitas vezes prefixados com "0" ou "0o" (por exemplo, 0755 ou 0o755). No contexto de texto, cada caractere pode ser representado por seu valor octal correspondente, geralmente usando três dígitos.
