Como usar:

MAIÚSCULAS: Converte todo o texto para caixa alta.

minúsculas: Converte todo o texto para caixa baixa.

Capitalizar: Converte a primeira letra de cada palavra para maiúscula.

iNVERTER: Inverte a capitalização de cada letra (maiúsculas viram minúsculas e vice-versa).
